# Diabetes Predictor

## Quick Start

```
git clone <this_repo>
cd <cloned_dir>

virtualenv -p python3 venv
source venv/bin/activate

./run requirements.install.dev
./run app.train
./run app.test
```

## Available Commands

- `./run pre_commit.init`
- `./run pre_commit.run_for_all`
- `./run requirements.compile`
- `./run requirements.install.dev`
- `./run requirements.install.prod`
- `./run app.train`
- `./run app.test`
- `./run app.tensorboard`
